# InnoPrompt Utility Platform ⚡

An enterprise-grade Prompt Engineering and Prototyping Platform that uses advanced dynamic prompt structures and fine-tuning to generate tailored system prompts across multiple production-ready engineering templates.

## 🚀 Live Production URL

👉 [https://innoprompt-studio-ab5afofftxscnuporrxdkz.streamlit.app/]

## 🎯 Project Overview & Core Features

* **Multi-Blueprint Engine**: Instantly drafts structured engineering system prompts across 3 discrete specialized domains.
* **Workspace Sandbox Presets**: Save, reload, and change customized prompt library structures instantly.
* **SaaS Interface Design**: Clean typography glassmorphic layout with custom deep-slate mesh gradient backdrops.
* **Real-time Live Compiler**: Tracks and displays the fully structural output compilation dynamically as you type.
* **Production Export Assets**: Instantly execute and run compiled prompt variants directly against live LLM sandboxes.

## 🛠️ Tech Stack & Architecture

* **Frontend UI Framework**: Streamlit (Python)
* **LLM Engine**: Google Gemini Pro / Flash Models
* **SDK**: Google GenAI Library (`google-genai`)
* **Environment Control**: Streamlit Secrets Management (`st.secrets`)
