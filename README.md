# 🚀 InnoPrompt Utility Platform

A high-performance, executive-grade Prompt Engineering and Prototyping Platform built with **Streamlit** and powered by the **Google Gemini API** (`google-genai`).

This platform is designed to help engineers construct, test, validate, and library-manage structured system prompts using industry-standard prompt patterns (Role, Context, Constraints, Few-Shot Examples, and Success Criteria).

---

## 💎 Features & Architecture

- **Premium UI/UX**: Custom-engineered glassmorphic interface featuring a deep slate mesh radial gradient background, reactive input focus glows, and polished typography.
- **Dynamic Structural Blueprints**: Pre-configured, high-fidelity prompt templates spanning:
  - **Cybersecurity**: SOC Incident Log Analysis
  - **Computer Science**: Multi-Core Parallel Code Optimization
  - **Electronics**: Passive Circuit Protection Design
- **Live Output Compiler**: Side-by-side active markdown compiler showing raw prompt structure as you type.
- **Local Sandbox Library**: Instant session-state storage to save, reload, or delete your customized prompt structures.
- **Gemini Playground**: Direct, full-screen model runner supporting native `google-genai` SDK implementation with fallback API key inputs.

---

## 🛠️ Tech Stack & Dependencies

- **Frontend/App Framework**: Streamlit (Python)
- **AI Integration**: `google-genai` SDK
- **Styling Engine**: Inline CSS Injection (HTML `unsafe_allow_html`)
- **State Management**: Streamlit Session State (`st.session_state`)
